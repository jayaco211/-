[유튜브 트렌드 분석기 사용 설명서]

안녕하세요! 이 프로그램은 유튜브 트렌드를 국가별, 영상 길이별로 분석해주는 도구입니다.

## 실행 방법
1. 받으신 폴더의 압축을 풉니다.
2. 폴더 안에 있는 **`run_app.bat`** 파일을 더블 클릭합니다.
   (이름이 `run_app`이고 톱니바퀴 모양 아이콘일 수 있습니다)
3. 검은 창이 뜨고 잠시 기다리면, 인터넷 창(브라우저)에서 프로그램이 실행됩니다.

## 주의 사항
* 이 프로그램을 실행하려면 컴퓨터에 **Python(파이썬)**이 설치되어 있어야 합니다.
* 만약 실행이 안 된다면, https://www.python.org/downloads/ 에서 최신 버전을 다운로드하여 설치해주세요. (설치 시 "Add Python to PATH" 체크 필수!)
* 처음 실행할 때는 필요한 파일을 설치하느라 시간이 조금 걸릴 수 있습니다.

## 사용 팁
* 일본 트렌드를 볼 때는 검색어를 한국어로 입력해도 자동으로 일본어로 번역해서 찾아줍니다. (예: '여행' -> '旅行')
* 하루 검색 횟수 제한(유튜브 정책)이 있으니 참고해주세요.

즐겁게 사용하세요!
